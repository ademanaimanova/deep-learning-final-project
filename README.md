# Deep Learning Final Project

## Project Title
Sentiment Analysis on Amazon Fine Food Reviews

##  Author: Naimanova Adema

##Project Overview
This project focuses on building and evaluating deep learning models for **sentiment analysis** using the Amazon Fine Food Reviews dataset.  
The goal is to classify customer reviews into **positive** and **negative** categories, compare baseline approaches with advanced deep learning models, and analyze performance.

## Repository Structure
- `project-proposal.md` — initial project proposal  
- `data/` — dataset instructions and subsets  
- `notebooks/` — exploratory analysis and experiments  
- `src/` — source code for models  
- `reports/` — weekly progress reports and final documentation  

## Objectives
- Prepare and document dataset setup  
- Perform exploratory data analysis (EDA)  
- Build baseline models (Logistic Regression, MLP)  
- Train deep learning models (LSTM, Transformer/BERT)  
- Evaluate results and interpret findings  
- Deliver final report and presentation
